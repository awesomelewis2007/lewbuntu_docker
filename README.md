# Lewbuntu

Lewbuntu is a docker image that has all the tools that I have created. It is based on ubuntu.

## Installed tools

- [Light builtins](https://www.github.com/awesomelewis2007/light_builtins)
- [Find](https://www.github.com/awesomelewis2007/find)
- [Search](https://www.github.com/awesomelewis2007/search)
- [Rust Hexdump](https://www.github.com/awesomelewis2007/rust_hexdump)
- [C_bench](https://www.github.com/awesomelewis2007/c_bench)
